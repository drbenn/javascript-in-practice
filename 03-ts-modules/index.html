TS - Modules

npm install typescript --save-dev

Create a TypeScript Configuration File: npx tsc --init



"npx tsc" is used to transform ts to js, it is helpful to put the output files in a seperate folder "dist"

So make the following changes to tsconfig.json
  • "outDir": "./dist", - all ts files will be compiled and placed into a seperate dist folder for easier organization
  • in index html now reference ----script type="module" src="dist/main.js"---- although main.ts is in the folder structure next to index.html, the output of main.js will be compiled to dist/main.js so the index.html must reference the compiled js file
  • If using modules will need to update tsconfig to the following es6 instead of es2016 and commonJs - "module": "es6" AND "module": "es6",

package script for nodemon should also change to compile both ts and scss on each save/change of scss/ts - "watch": "nodemon -e ts,scss -x \"npm run css && npx tsc\""


in terminal: npm run watch and then in vsc start live server
